# 项目地图

本文件说明 `D:\1Folder\语言学习-lang-drill-skill` 的目录结构、核心入口和长期维护信息。

## 项目目标

- 维护通用语言学习刷题 skill：`lang-drill-coach`。
- 让智能体根据考纲、学习背景、学习进度和复习算法，生成语言学习测验并维护长期复习状态。
- 当前优先支持日语和英语；日语已有可复用考纲资产，英语在用户确定考试目标后导入。

## 根目录

- `AGENTS.md`：项目级规则，开始实质工作前先读。
- `项目地图.md`：当前文件，记录长期维护信息。
- `.gitignore`：Git 忽略规则，必须忽略 `.env`、日志和临时目录。
- `.env.example`：环境变量假账本，只放占位值。
- `LICENSE`：PolyForm Noncommercial License 1.0.0，禁止未经授权的商业使用。
- `COMMERCIAL_LICENSE.md`：商业授权说明。
- `PUBLISHING.md`：GitHub 与 skill 发布检查清单。
- `README.md`：项目介绍、使用入口与授权说明。

## `data/`

- `data/study.db`：主 SQLite 数据库。保存词汇、语法、会话、题目、作答、错题、资料索引等结构化状态。
- `data/background/student_profile.md`：学习者档案模板；首次使用时补充目标语言、考试目标、截止时间、题量、偏好等信息。
- `data/intake/`：用户当日导入原料和中间输入，当前已清空。
- `data/kb/`：考纲、词表、真题索引和资料源。
- `data/kb/cjt4/`：大学日语四级 2023 考纲词汇、语法、模拟蓝图和历史索引。
- `data/kb/gaokao-japanese/`：高中日语 2020 课程/词法资源。
- `data/kb/material-inbox/`：待导入的本地资料收件箱。

## `doc/`

- `doc/项目局部规则.md`：本项目的通用初始化、组题、复习和资料边界规则。
- `doc/进展记录.md`：阶段快照、产出和错误汇报。
- `doc/错题集.md`：正式会话答错题目的可读错题本。
- `doc/习题日记/`：按实际使用题目重建的学习回放，当前已清空个人历史。
- `doc/当日总结/`：每日正式会话收口后的总结目录，当前已清空个人历史。
- `doc/真题/`：真题资料的人读索引目录。

## `scripts/`

日常链路：

1. `init_today.py`：生成通用初始化面板，缺少建档信息时返回待询问问题。
2. `import_vocab.py`：导入词汇。
3. `import_grammar.py`：导入语法。
4. `select_session_content.py`：选择当天新学、到期、复习中、已掌握和错题回流候选。
5. `extract_background_candidates.py`：从候选池抽取背景锚点，辅助 agent 生成自然语境。
6. `persist_authored_session.py`：将 agent 编写的整套题单正式落库。
7. `session_status.py`：查询当前活跃会话、进度和下一道待做题。
8. `grade_answer.py`：判题并即时回写作答、掌握度、错题和日记。
9. `reconcile_session.py`：会话完成后校准长期熟练度。
10. `audit_study_day.py`：审计当天数据状态和覆盖度。
11. `sync_progress_snapshot.py`：追加阶段进展快照。
12. `publish_skill.py`：同步 `skills/lang-drill-coach/` 到 `D:\2Folder\skills`。

资料维护：

- `build_official_kb.py`：构建已保留的日语官方知识库。
- `build_cjt4_historical_manifest.py`：重建大学日语四级历史真题索引。
- `download_cjt4_historical_papers.py`：下载/更新大学日语四级历史页面快照。
- `import_materials.py`：索引 `data/kb/material-inbox/` 中的本地资料。

## `skills/`

- `skills/lang-drill-coach/SKILL.md`：通用语言刷题 skill 主文档。
- `skills/lang-drill-coach/agents/openai.yaml`：skill 广场/客户端展示元数据。
- `skills/lang-drill-coach/references/commands.md`：常用命令。
- `skills/lang-drill-coach/references/workflow.md`：初始化、日常刷题和恢复流程。

## 运行方式

```powershell
py .\scripts\init_today.py
py .\scripts\select_session_content.py --target-minutes 35
py .\scripts\publish_skill.py
```

## 当前功能清单

- 通用建档初始化：`scripts/init_today.py`，状态：可用。
- 词汇/语法导入：`scripts/import_vocab.py`、`scripts/import_grammar.py`，状态：可用。
- 复习候选选择：`scripts/select_session_content.py`，状态：沿用日语项目逻辑，已改为非用户考纲兜底。
- Agent 成题与落库：`scripts/persist_authored_session.py`，状态：可用。
- 逐题判题回写：`scripts/grade_answer.py`，状态：可用。
- Skill 发布：`scripts/publish_skill.py`，状态：发布到 `D:\2Folder\skills\lang-drill-coach`。
- 非商用授权：`LICENSE`、`NOTICE`、`COMMERCIAL_LICENSE.md`，状态：可用。

## 已知限制

- 英语考试考纲尚未内置，需要用户确认考试目标后导入。
- 背景抽取和部分题型偏好继承自日语项目，后续可继续抽象为语言/考试配置。
- 历史真题默认只做索引和参考，不保证完整正文可发布。
