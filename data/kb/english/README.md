# English Knowledge Base

This directory is reserved for English exam syllabi and reference materials.

Create one subdirectory or file set per exam after the learner chooses a target, for example:

- `cet4/official_vocab_YYYY.json`
- `ielts/official_task_types_YYYY.json`
- `toefl/official_grammar_YYYY.json`

Record the source name and year in every structured asset.
