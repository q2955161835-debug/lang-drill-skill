# 高考日语知识库

本目录用于存放高考日语基础支撑资料，作为大学日语四级训练的补底层。

当前内容：
- `official_vocab_2020.json/.csv`：根据《普通高中日语课程标准（2017年版2020年修订）》附录4 自动提取的完整词汇表。
- `official_grammar_2020.json/.csv`：根据《普通高中日语课程标准（2017年版2020年修订）》附录3 自动提取的语法表。
- `official_papers_index.json`：2024 年高考综合改革适应性测试日语科官方页面索引。
- `seed_vocab.json`：启动用基础词汇样本。
- `seed_grammar.json`：启动用基础语法样本。

当前统计口径：
- 结构化词汇：1000 条，和官方课程标准词汇表数量一致。
- 结构化语法：118 项自动提取条目。
- 官方公开题源索引：1 项（2024 适应性测试页面）。

后续可扩展：
- 课程标准拆分词表
- 高频语法点与易混点
- 历年高考日语真题索引与导入位
- 阅读与写作模板
