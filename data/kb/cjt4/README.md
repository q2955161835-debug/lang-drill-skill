# 大学日语四级知识库

本目录用于存放大学日语四级知识库与考试结构资料。

当前内容：
- `exam_overview.md`：考试结构与使用说明。
- `official_vocab_2023.json/.csv`：根据《全国大学日语四、六级考试大纲（2023年版）》自动提取的词汇表。
- `official_grammar_2023.json/.csv`：根据《全国大学日语四、六级考试大纲（2023年版）》自动提取的语法项目表。
- `official_papers_index.json`：官方四级样卷、听力文字稿、标准答案索引。
- `seed_vocab.json`：启动用大学日语四级词汇样本。
- `seed_grammar.json`：启动用大学日语四级语法样本。

当前统计口径：
- 结构化词汇：4694 条自动提取项，外加原始官方 PDF 原件作为兜底来源。
- 结构化语法：276 项，和官方大纲列项数量一致。
- 官方公开题源索引：3 项（样卷、听力文字稿、标准答案）。

后续可扩展：
- 按题型整理的题目模板
- 历年真题索引与导入位
- 高频阅读主题
- 翻译与写作常见表达
